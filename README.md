# Tabela Hash por Encadeamento Separado

Implementação em Java de tabela hash com encadeamento separado.

### Arquivos principais
- `Main.java` — orquestra experimentos e imprime CSV (saída padrão).
- `FuncoesHash.java` — implementa H_DIV, H_MUL, H_FOLD.
- `Registro.java` — nó simples para listas encadeadas.

### Como compilar
```bash
javac Main.java FuncoesHash.java Registro.java
